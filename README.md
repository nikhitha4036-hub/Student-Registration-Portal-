📚 Student Registration Portal
📌 Project Overview

The Student Registration Portal is a responsive web application built using HTML, CSS, JavaScript, and Bootstrap.
It allows students to register through a multi-step form with validations, progress tracking, and user-friendly UI features.

🚀 Features
🧾 Multi-step registration form
📊 Progress bar for step tracking
📧 Email validation
📱 Mobile number validation
🆔 Aadhaar number validation
🔒 Password strength checker
👁️ Show/Hide password toggle
🌙 Dark mode toggle
📂 Dynamic course & department dropdown
💾 Auto-save form data (LocalStorage)
🖼️ Profile image upload validation
🎉 Success toast notification
🔄 Form reset & redirection after submission

🛠️ Technologies Used
HTML5
CSS3
JavaScript (ES6)
Bootstrap 5
